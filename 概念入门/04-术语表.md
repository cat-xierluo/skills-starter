# 术语表

本文档收录 AI 编程领域的常用术语，帮助你快速入门。

## A

| 术语 | 英文 | 解释 |
|------|------|------|
| Agent | Agent | AI 智能体，能自主决策和执行任务的 AI 程序 |
| API | Application Programming Interface | 接口，程序之间对话的「电话」 |
| Artificial Intelligence | AI | 人工智能，让机器具有人类智能 |

## B

| 术语 | 英文 | 解释 |
|------|------|------|
| Branch | Branch | 分支，代码开发的平行工作线 |
| Bug | Bug | 程序错误、缺陷 |

## C

| 术语 | 英文 | 解释 |
|------|------|------|
| CLI | Command Line Interface | 命令行界面，文字输入的操作系统交互方式 |
| Clone | Clone | 克隆，把远程仓库复制到本地 |
| Cloud | Cloud | 云，基于互联网的远程服务器 |
| Commit | Commit | 提交，代码的存档点 |
| Container | Container | 容器，封装软件运行环境的轻量级虚拟化技术 |

## D

| 术语 | 英文 | 解释 |
|------|------|------|
| Debug | Debug | 调试，修复程序错误 |
| Deploy | Deploy | 部署，把代码发布到服务器上线运行 |
| Developer | Developer | 开发者，写程序的人 |
