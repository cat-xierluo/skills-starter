# 参考文档目录

本目录用于存放当前 Skill 的详细参考资料。

适合放这里的内容：

- API 说明
- 边界条件说明
- Prompt / 输出格式样例
- 第三方服务接入说明
- 不适合塞进 `SKILL.md` 的长文档

不适合放这里的内容：

- 当前任务状态
- 变更历史
- 路线图

这些内容应该分别写入：

- `TASKS.md`
- `DECISIONS.md`
- `ROADMAP.md`
- `CHANGELOG.md`
