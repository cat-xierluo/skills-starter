# 03 基于模板创建

没有现成方案时，从模板开始。

## 使用 skill-template

```bash
# 克隆模板（如果还没有）
git clone https://github.com/cat-xierluo/skill-template.git <skill-name>

# 或者复制模板目录
cp -r skill-template/ ../<skill-name>
```

模板地址：`https://github.com/cat-xierluo/skill-template`

## SKILL.md 结构解析

```markdown
# Skill 名称

[简短描述：这个 Skill 做什么]

## 触发条件
[什么情况下调用]

## 工具
[使用哪些工具]

## 参数
[输入参数说明]

## 输出
[返回结果说明]

## 示例
[使用示例]
```

**每个字段都要写，但不要废话。**

## 如何命名 Skill

- **命名格式**：`行为-对象` 或 `对象-行为`
  - ✅ `github-issues`（行为-对象）
  - ✅ `image-understand`（对象-行为）
  - ❌ `do-something-useful`
- **用英文**（社区通用语言）
- **简短**：1-2 个单词最佳
- **清晰**：不看描述也能猜到用途

## 创建步骤

1. **复制模板**：`cp -r skill-template/ ../<skill-name>`
2. **命名**：目录名即 Skill 名
3. **写 SKILL.md**：描述清楚做什么、怎么用
4. **实现代码**：在 `scripts/` 下写核心逻辑
5. **本地测试**：跑通基本流程
6. **调试优化**：见 04 调试与发布

---

**原则**：先让它跑起来，再优化。先完成，再完美。
